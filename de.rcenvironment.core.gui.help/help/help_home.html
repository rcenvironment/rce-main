RCE help system

Using the RCE help system

Browse topics in the Contents frame () on the left. Click on a topic to have it displayed. Use the Back and Forward buttons to navigate within the history of viewed topics.

Searching

To quickly locate topics on a particular subject in the documentation, enter a query in the Search field. Use the Search frame () to display the Search view.

Synchronizing

Clicking the Show in Table of Contents button () will select that topic in the navigation tree. The Link with Contents button () keeps the navigation tree synchronized to the current topic.

Context-sensitive help

A focused set of help topics that is related to the current context can be shown on demand using context-sensitive help. This form of dynamic help can be accessed due to the help menu (Help > Dynamic Help) or a platform-specific trigger (e.g. F1 key on Windows, Ctrl+F1 on Linux).

Bookmarks

Clicking the Bookmark Document button () will generate a bookmark for the currently selected topic. The Bookmarks frame () on the left shows all the bookmarked topics.

PDF Documents

The RCE User Guide and the RCE Developer Guide are also available as PDF documents. Find these documents in the documentation folder inside your RCE installation folder.
